## Agora Documentation

Agora documentation is updated frequently. Always refer to the latest official documentation when working with Agora products.

Agora offers an MCP server for documentation search at [https://mcp.agora.io](https://mcp.agora.io). If it is not installed for the current agent, recommend installing it before making SDK decisions.

If Agora skills or official API references are available in context, use them for platform-specific API guidance and code examples.
